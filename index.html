Data Science

Introduction to Data Science here!

Data science study where data comes from and what it means. This information is then analysed to see how it can be made useful. Data science is made up of many different fields including math, statistics, advanced computing, visualization, data engineering, and machine learning. The purpose of data science is to find hidden patterns and trends inside of data using the different fields that make up data science.

Machine Learning

Introduction to machine learning!

Machine learning is defined as the ability for computer systems to learn by themselves without any real time input. Machine learning does this using algorithms to find trends and patterns of past outcomes to predict future outcomes. Machine learning is one of the fields that make up data science. Machine learning algorithms use MASSIVE amounts of information and run millions of trials in order to very accurately predict ideal outcomes for a given scenario.
